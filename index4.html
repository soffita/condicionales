Condicionales
Sintaxis
La formula con la que vamos a hablar hoy es la siguiente: =SI(C3:C7>=3;"Gano";"Perdio"). El "SI" se utilizando para comprobar si la condición se cumple, "C3:C7" es el valor que se debe evaluar para ver si la condición pedida se cumple. ">=3" es la condición que se pide para realizar la acción, "Gano" y "Perdio" son los valores que se van a dar si la accion es verdadera o falsa.
¿Como se realizan?
Debemos pensar en la accion que se va a hacer, tener las valores para hacerla y realizar la estructura de la formula de una buena forma.
